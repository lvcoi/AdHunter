# Privacy Policy for Ad Hunt

**Effective Date: April 2, 2026**

## Introduction

This Privacy Policy describes how the Ad Hunt Chrome extension ("the Extension") handles user data. We are committed to protecting your privacy and being transparent about our data practices.

## Data Collection and Storage

The Extension does **not** collect, transmit, or share any personal data. All data generated by the Extension remains entirely local to your browser.

The Extension stores the following data locally using Chrome's built-in storage APIs:

- **High Scores** — Your game high scores are stored using `chrome.storage.sync`, which allows them to sync across your Chrome browsers if you are signed in to Chrome. This data never leaves Google's Chrome sync infrastructure and is not accessible to us or any third party.
- **Element Vault Data** — Metadata about DOM elements you have removed (such as tag names, class names, and dimensions) is stored using `chrome.storage.session`. This data exists only for the duration of your browser session and is automatically cleared when the browser is closed.

## Permissions and Their Purpose

The Extension uses the following browser permissions solely to provide its core functionality:

- **`declarativeNetRequest`** — Used to block requests to known ad network domains (such as `doubleclick.net`, `googlesyndication.com`, and `googleadservices.com`). Blocking is performed entirely by the browser using declarative rules. No network traffic is intercepted, recorded, or transmitted.
- **Content Script (all pages)** — A content script runs on web pages to enable the duck hunting game overlay and the element inspection tool. The content script operates locally within your browser and does not send any page content or user activity to external servers.

## Data We Do NOT Collect

To be explicit, the Extension does **not**:

- Collect personally identifiable information (PII)
- Track your browsing history or behavior
- Use analytics, telemetry, or tracking tools of any kind
- Transmit any data to external servers
- Store cookies or use fingerprinting techniques
- Access or read the content of web pages for any purpose other than enabling the in-page game and element inspector

## Third-Party Sharing

We do not share any data with third parties. There is no data to share because the Extension does not collect any.

## Data Security

Because all data is stored locally within your browser using Chrome's built-in storage APIs, your data is protected by Chrome's own security model. No data is transmitted over the network to any server we operate, which eliminates the risk of server-side data breaches.

## Children's Privacy

The Extension does not knowingly collect any personal information from anyone, including children under the age of 13.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by updating the "Effective Date" at the top of this document. The updated policy will be made available alongside the Extension listing on the Chrome Web Store. We encourage you to review this Privacy Policy periodically.
