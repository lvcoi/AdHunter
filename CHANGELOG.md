# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-04-02

### Added
- Rebrand to **Ad Hunt** with retro duck-shooting game mode integration.
- Rearchitected side panel with game over screen and high scores.

### Fixed
- Gameplay issues resolved during rebrand.

## [0.3.0]

### Added
- Duck Hunt mini-game that replaces empty ad slots with interactive targets and score tracking.
- Tiered duck scoring, disintegration animations, and enhanced audio feedback.
- Sequential ad processing with free-roaming ducks and visual effects.

## [0.2.0]

### Added
- New animated highlight styles with default selection set to core-spin.
- Expanded animation library with 10 levels; migrated storage to session API.
- Content script registered in manifest.
- Advanced Color Palette (US-001).
- Advanced Gradient Builder (US-002).
- Custom Border Builder UI (US-002).
- Save Custom Profile support (US-003).

### Changed
- Replaced legacy highlight styles with advanced CSS animations and effects.
- Refined element highlight styles into a minimalist design system.
- Updated and expanded CSS overlay styles with new animations.

### Removed
- Obsolete PRD documentation files.

## [0.1.0]

### Added
- Initial AdHunter Chrome Extension files.
- Default border styles (US-001).
